# checked-clock
A clock (custom element) that uses other checked-elements as bits to draw numbers

Usage:
```html
<checked-clock>
  <input type="checkbox">
</checked-clock>

<checked-clock>
  <my-switch></my-switch>
</checked-clock>

<checked-clock>
  <input type="radio">
</checked-clock>
```
